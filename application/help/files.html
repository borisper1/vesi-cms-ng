Gestione file e immagini

Per caricare, spostare o eliminare file e immagini dal sito è possibile utilizzare il gestore di file e immagini integrato nell’interfaccia amministrativa al posto dell’accesso FTP.

La sezione File rappresenta la cartella files/ dell’installazione di vesi-cms, la sezione Immagini rappresenta la cartella img/.

Per eliminare, spostare, copiare o rinominare un elemento selezionarlo e premere sull’azione desiderata nella barra degli strumenti. È anche possibile premere sull’icona accanto al nome dell’elemento e scegliere l’azione desiderata; inoltre selezionando più elementi le azioni verranno eseguite su tutti gli elementi selezionati se si usa i pulsanti della barra degli strumenti.

Per scaricare un elemento selezionalo e premere su Scarica nella barra degli strumenti oppure fare click sull’icona accanto al suo nome. Qualora questa icona non venga visualizzata fare clic sul nome dell’elemento per scaricarlo. Le cartelle saranno scaricate come file compressi (formato ZIP), se si seleziona più elementi verrà scaricato un file ZIP contenente gli elementi selezionati.

Premendo sul nome di un elemento si accederà al suo contenuto se questo elemento è una cartella, se l’elemento è un file si visualizzerà una sua anteprima per i tipi di file supportati (file PDF, immagini, file audio e video) oppure verrà scaricato il file.

Per tornare alla cartella precedente utilizzare la barra di navigazione posta sotto la barra degli strumenti, non utilizzare il pulsante indietro del browser.

Premendo il pulsante Nuovo sarà possibile creare una nuova cartella e caricare dei nuovi file. Nella sezione File è possibile caricare ogni tipo di file di qualsiasi dimensione ad eccezione dei file con estensione php, pl, cgi, sh, fcgi, inc, py, bat, exe, cmd, dll e ocx. Nella sezione Immagini è possibile caricare solo file di immagine con estensione jpg, jpeg, png, gif, bmp e svg.

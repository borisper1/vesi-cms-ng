Editor del contenuto file-lister

Questo componente permette di visualizzare l’elenco dei file contenuti in una cartella, offrendo la possibilità di scaricarli. Può essere usato per presentare documenti amministrativi. Nell’editor va indicato il percorso relativo alla radice del sito della cartella che contiene i file (generalmente files/…), che vengono visualizzati in ordine alfabetico.

Per visualizzare un’anteprima di come saranno visualizzati i file premere su .

Editor del contenuto link

Questo componente permette di inserire un link e personalizzare facilmente il suo aspetto. È necessario inserire un link assoluto. È possibile scegliere se il link si deve aprire nella stessa finestra o in una nuova scheda, e inoltre è possibile assegnare classi Bootstrap/CSS al link (ad esempio btn, btn-primary, btn-block, ecc. per ottenere un aspetto simile a un pulsante).

Editor del contenuto html-field

Questo componente permette di inserire codice HTML personalizzato all’interno della pagina. È il componente più flessibile che permette di sfruttare tutte le funzionalità del software. L’inclusione di codice JavaScript è supportata, ma possono essere inclusi solo file che devono essere caricati tramite FTP, i tag <script> e alcuni altri tag sono vietati per motivi di sicurezza. L’editor HTML supporta la correzione degli errori e syntax highlighting
